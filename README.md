
# Description 

本專題為資料可視化工具的實作，將載入的特定JSON格式資料集以柱狀圖呈現，以下拉式選單決定某特定時間的資料。


## JSON 資料格式

```javascript
{"source":"XX統計處",
 "data":{
     "time1":{"key1":value,
              "key2":value,
              "key3":value},
    "time2":{"key1":value,
             "key2":value,
             "key3":value}
 }}
```


## API Reference


#### add(num1, num2)

Takes two numbers and returns the sum.
